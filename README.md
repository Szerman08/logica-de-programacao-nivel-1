# Lógica de Programação — Nível 1

Exercícios introdutórios de lógica de programação, cobrindo os fundamentos do raciocínio computacional.

## Sobre

Repositório com os exercícios do nível 1 do curso de lógica de programação. Abrange conceitos básicos como variáveis, condicionais, laços de repetição e classificações simples.

## Tecnologias

- Lógica de Programação (Pseudocódigo / Algoritmos)

## Conteúdo

- 30 exercícios progressivos (exercício 1 ao 30)
- Classificação de idade
- Estruturas de decisão e repetição
- Problemas do cotidiano resolvidos com algoritmos
